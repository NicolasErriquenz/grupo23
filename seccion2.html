¿Que es CSS?


CSS (Cascade Style Sheet) es un lenguaje de diseño que define la apariencia de diversos elementos de un documento HTML.Se ocupa de la estética y el aspecto.


¿Como lo incorporamos?


CSS externo: En el head del documento HTML, debemos incluir una referencia al archivo .css dentro del elemento link.




Tambien podemos incluir el CSS de manera interna o en linea, tal como se explica en la siguiente clase.

Estructura


Selector: Marca el elemento sobre el cual vamos a aplicar una regla de estilo.


Propiedad y Valor: Especifica la caracteristica que voy a modificar de un elemento y el valor que adoptara.


Declaración: Señala el estilo que le daremos al selector.
