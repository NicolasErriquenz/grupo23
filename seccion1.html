¿Que es HTML?


HTML (Hyper Text Markup Language)es el componente mas basico de la web, define el significado y la estructura del contenido web.


¿Como se utiliza?


HTML utiliza etiquetas para identificar texto, imagenes y otro contenido para mostrarlo en un navegador web.


Estructura


Las etiquetas ayudan al navegador a entender su significado para mostrarlo en pantalla y ayuda a los buscadores a reconocer el contenido e identificar una estructura determinada; debemos respetarlas.
